Описание:
	Название: financial
	Предметная область: пользователи и их списки расходов, доходов, кредитов, счетов
Данные:
	Общие ограничения: цисла должны быть целыми неотрицательными до 10^10 включительно;
	Ограничения для пользователей: логин должен быть уникальным с длиной не более 50 символов;
		пароль должен быть длиной не более 300 символов;
	Ограничения для наименований категорий: названия категорий должны быть уникальными;
		длина названия до 50 символов;
	Ограничения на записи расходов, доходов: дата должна быть не ранее текущей даты;
Пользовательские роли:
	"потребитель": человек, заинтересованный в контроле своих денежных операций.
		Отвественность за корректное представление данных для верных расчетов.
		Количество уникальных пользователей в это роли не ограничено.
UI/API:
	для представления информации реализовано веб-приложение с графическим представлением.
Технологии разработки:
	Язык программирования: Python3.9
	СУБД: PostgreSQL 15
Тестирование представлено в файле tests.py
